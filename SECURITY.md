# Security Policy

We currently support and maintain the following versions of our original projects:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

We take the security of our projects seriously. If you discover a vulnerability, please report it through our Bug Bounty Program.

### How to Report
- **Email**: [onlybugs05@gmail.com](mailto:onlybugs05@gmail.com)
- **Response Time**: We aim to acknowledge reports within 48 hours.

### Bug Bounty Program
We offer rewards for qualifying vulnerabilities found in our **original** repositories (excluding forked repos):

- 🎁 **Exclusive Swag**
- 💵 **Cash Bounties**: $100 USD - $1,000 USD

### Scope & Terms
- Only original repositories under the `@onlybugs05` organization are in scope.
- Forked repositories are strictly out of scope.
- Bounty payments and amounts are at the sole discretion of the project owner.
- Rewards are subject to fund availability.

Please do not disclose the vulnerability publicly until it has been resolved and you have received confirmation from us.
